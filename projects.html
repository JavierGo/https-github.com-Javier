Programmable Circuits Project

Challenge 0

Orientation Challenge: Wire a simple circuit with an LED. Recreate this circuit using a total of four different orientations proving your mastery of circuit fundamentals. After completing this challenge each student will take a quiz with images of circuits where you must identify if the LED will illuminate (or not).

Challenge 1

Blink Challenge: Program an LED to blink in a unique pattern lasting at least 5 seconds; use the fewest lines of code possible.

Code for all challenges.

Challenge 2

User Input Challenge: Program a circuit that receives input from the user and produces a visible output in your circuit. Input can be taken from the keyboard or by using buttons/sensors from your sensor kit.

For challenge 2 we decided to program our circuit to flash Andy's (our computer science teacher) full name in Morse Code. To do this we made comands using python code. Challenge 3

Light Show Challenge: Program a light show that accompanies a song played from your Raspberry Pi. Your light show should match the beat/rhythm of the chosen/composed song. All projects will be displayed at a final showcase event during class.
